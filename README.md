# Suite Landing Page

A responsive marketing landing page built with HTML and CSS, featuring interactive states and pixel-perfect design implementation from Figma.

## Features

- **Fully Responsive**: Mobile (375px), Tablet (768px), Desktop (1440px)
- **Interactive States**: Focus and hover effects on buttons and links
- **Pixel-Perfect Design**: Accurate implementation from Figma specifications
- **Semantic HTML**: Proper structure with accessibility considerations
- **Custom CSS**: No frameworks, pure CSS with media queries

## Design Elements

- Hero section with curved line accent and gradient phone mockup
- Statistics showcase (2K+ companies, 8 languages, 1.2M leads)
- Testimonial section with blurred gradient background effect
- Social media links with focus indicators
- Dynamic button states (focus rings, gradient hover effects)

## Typography

- **Font**: Epilogue (Google Fonts)
- **Weights**: Regular (400), Bold (700)
- **Responsive sizing**: 38px - 72px for headings

## Color Palette

- Primary Dark: `#172339`
- Background Cream: `#FAF8F6`
- Secondary Beige: `#F3EDE7`
- Text Gray: `#49566D`
- Accent Purple: `#CB30E3`

## Interactive States

### Focus States (Desktop)
- Double-ring focus indicators (3px inner, 6px outer)
- Yellow highlight on heading text when hero button focused
- Social media icons with visible focus rings

### Hover States (Desktop)
- Navbar button: Dark background with white text
- Hero button: Purple-to-orange gradient background
- Social icons: 50% opacity
- Heading text: Yellow highlight effect

## Browser Support

Modern browsers supporting CSS Grid, Flexbox, and CSS custom properties.
